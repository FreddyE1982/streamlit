This repo contains the code of "streamlit" a python library for creating GUIs for python apps.
The agents task is to iteratively modify streamlit to enable testing of any app that uses a streamlit gui via pytest.
There must be a way for tests to simulate the usage of ALL gui interactive gui elements (activating, deactivating, entering text, getting values, setting values).
This must be fully integrated with and into the streamlit code.

Note that the code of the library is extensive, so you will not be able to make all nessecary modifikations in on go before agent timeout, you have to modify the code iteratively.
